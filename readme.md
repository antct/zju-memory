## Run

> Fill in `memory/config.ini` first

```bash
cd memory
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py
```

```
docker build -t memory:1.0 .
docker run -it --rm -p 5000:5000 memory:1.0
```

> View it in your browser → http://localhost:5000/
